# GA
　GA(Genetic Algorithm)は遺伝的アルゴリズムといい、生物界の進化の仕組みを模倣した解探索手法です。本コードはeclipseのC++でGAを実行するためのプログラムです。

# データセット
　本研究では、初期世代の個体を最終世代までに最小化させる実験を行いました。データ項目数は4で、項目内の個体数は300です。  
　＊最小化問題  
 　　・面積と初期費用  
 　　・築年数  
 　　・向き(向きに関しては個体数276)  
 　　・最寄り徒歩何分
